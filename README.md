# ClearSky
Описание проекта:

Программный продукт для владельцев дронов DJI Mini 2 - это специализированное приложение для определения оптимальных погодных условий для безопасных и эффективных полетов.

Главная идея проекта:

Обеспечить пользователей дронов необходимыми инструментами для анализа погодных условий, минимизируя риски, связанные с неблагоприятной погодой. Это позволит владельцам сосредоточиться на управлении и эксплуатации дрона, не беспокоясь о погодных препятствиях.

------------------------------------------------------------------------------------------------------------

Для создания приложения мы используем следующие библиотеки:

- tkinter и ttk: Позволяют создавать графические интерфейсы с элементами управления, такими как окна и кнопки.

- requests: Используется для выполнения HTTP-запросов и получения данных из интернета.

- datetime: Помогает работать с датами и временем, что важно для временного анализа.

- os: Обеспечивает взаимодействие с операционной системой, включая управление файлами.

- PIL и ImageTk: Предоставляют средства для обработки и отображения изображений в приложении.

Эти библиотеки обеспечивают функциональность и удобство использования нашего приложения.
